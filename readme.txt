== Description ==

Simple Disable Comments is a powerful WordPress plugin designed to give website administrators full control over comments on their WordPress websites. With this plugin, you can easily manage and customize the commenting system to suit your website's needs, whether you want to completely disable comments globally or on specific post types.

== Features ==

**Full Control:** The plugin provides website administrators with complete control over the comment section on their WordPress websites.

**Customization Options:** Users can easily manage and customize the commenting system to suit their website's needs.

**Global Comment Disabling:** The plugin allows administrators to disable comments globally across their entire website.

**Selective Comment Disabling:** Users can choose to disable comments on specific post types, such as pages or attachments, while keeping them enabled on other types of content.

**Moderation and Regulation:** Simple Disable Comments enables effective moderation and regulation of the interactions happening within the comment section.

**Spam Prevention:** The plugin helps prevent unwanted comments or spam from being displayed on the website.

**Appearance Customization:** Users have the ability to customize the appearance and behavior of the comment section.

**Streamlined User Experience:** The plugin allows administrators to hide existing comments or prevent new ones from being submitted, providing a streamlined user experience.

**Compatibility:** Simple Disable Comments is compatible with various WordPress themes and plugins, ensuring seamless integration and maximum functionality.

== Frequently Asked Questions ==

= Is this a free plugin? =

Yes it's a free plugin.

= Can I disable comments globally on my entire WordPress website? =

Yes, with the Simple Disable Comments plugin, you can easily disable comments globally across your entire website. This means that no comments will be displayed on any posts or pages.

= Can I disable comments on specific post types only? =

Absolutely! Simple Disable Comments allows you to selectively disable comments on specific post types, such as pages or attachments. You have the flexibility to choose which types of content should have comments disabled while keeping them enabled on others.

= Will existing comments be hidden if I disable comments using this plugin? =

Yes, when you disable comments using Simple Disable Comments, existing comments will be hidden from view. This helps ensure that your website's comment section reflects your desired configuration.

= Is this plugin compatible with other WordPress themes and plugins? = 

Yes, Simple Disable Comments is designed to be compatible with various WordPress themes and plugins. This ensures seamless integration and maximum functionality when using the plugin alongside other tools on your website.

= How user-friendly is the Simple Disable Comments plugin? =

The Simple Disable Comments plugin is designed to be user-friendly and easy to use. Its intuitive interface makes it simple for website administrators to manage and customize their comment section without requiring extensive technical knowledge.
